## User Story

As a new contributor,
I want a clear contributing guide,
so that I can effectively contribute to the project following established standards.

-**And**prop naming patterns are specified

-**And**accessibility requirements are listed

-**Given**contributor is ready to start coding

-**When**they follow the workflow guide

-**Then**setup steps are provided

-**And**testing workflow is documented

-**And**PR submission process is clear

-**And** review process is explained
